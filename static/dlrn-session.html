DLRN setup steps

1 Install mock

1.1 Install and Setup

dnf install mock

1.1.1 Add user to mock group

usermod -a -G mock [User name]

1.1.2 dlrn.cfg

wget https://gist.githubusercontent.com/lukehinds/856cb55b8f5b480d9e641f8812749178/raw/97f813e30c89090247e5eb4d5bdca2c087b377a1/dlrn.cfg

sudo mv dlrn.cfg /etc/mock

2 Install and run DLRN

  • Clone the repository git clone https://github.com/openstack-packages/DLRN
  • Follow the steps in the README.rst file to configure it (if you're already using mock, your user will be part of the mock group)
  • On another directory, git clone https://github.com/redhat-openstack/rdoinfo
  • Edit the rdotools/rdo.yml file in the repo and add a block like this: 
project: networking-bagpipe
  conf: rpmfactory-lib
  name: python-networking-bagpipe
  upstream: <upstream git url>

Example RDO.yml

  • Then, create a directory DLRN/data/python-networking-bagpipe_distro and put your spec file in there
  • finally, you can test the build with the following command-line: 
dlrn --config-file projects.ini --package-name python-networking-bagpipe --dev --info-repo <path to the rdoinfo directory>

For example: 

dlrn --config-file projects.ini --package-name python-networking-bagpipe --dev --info-repo ../rdoinfo
